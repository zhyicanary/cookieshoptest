# CookieShopTest

这是一个用于演示和测试的小型饼干店（Cookie Shop）项目骨架，包含基本说明和快速上手指南。

## 描述
示例项目，便于展示项目结构、启动流程与贡献规范。

## 快速开始
1. 克隆仓库或在本地创建项目目录。
2. 进入项目目录并安装依赖（示例以 Node 为例）：
   npm install
3. 启动（示例）：
   npm start

## 本地开发
- 使用 feature/* 分支进行功能开发。
- 提交前请运行测试与格式化工具。
